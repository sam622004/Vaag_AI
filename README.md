# Vaagai App

Vaagai is a modular Flutter + AI application designed to empower farmers and agricultural stakeholders.  
It integrates four key modules:

- 🌾 **Schemes Explorer** – Browse government schemes and benefits.
- 🧪 **Plant Disease Detection** – AI/ML powered detection using trained models.
- ☁️ **Weather Notifier** – Real-time weather alerts for farming regions.
- 💹 **AgMarket Price Dashboard** – Live commodity prices from Agmarknet API.

## Features
- Dynamic dropdown filters for commodities and markets
- Live API integration with mock fallback
- Clean, responsive Flutter UI
- Python ML backend for plant disease detection

## Tech Stack
- Flutter (UI)
- Dart (business logic)
- Python (ML model integration)
- REST APIs (Agmarknet, Weather)
